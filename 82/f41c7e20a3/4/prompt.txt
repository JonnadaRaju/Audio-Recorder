PHASE 2 — Video Processing Pipeline

Implement a processing pipeline:

Step 1
Extract audio from video using FFmpeg.

Example:

Step 2
Send extracted audio to Sarvam AI speech-to-text API.

Store transcript in database.

Step 3
Generate transcript embeddings using embedding model.

Store embeddings in pgvector.

Step 4
Generate summary using Groq LLM.

Store summary.

Add endpoints:
